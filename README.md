## How to Use

1. **Login or Register**:
   - Login with the default admin account:
     - **Username**: `admin`
     - **Password**: `1234`
   - Or register a new user.

2. **Message Encryption/Decryption**:
   - Enter the text in the input box.
   - Provide the secret key (`1234` by default).
   - Click `Encrypt` or `Decrypt`.
   - View the output text and character frequency plot.

3. **File Encryption/Decryption**:
   - Select the file to encrypt or decrypt.
   - The application saves the encrypted or decrypted file in the same directory.

4. **Reset**:
   - Use the `RESET` button to clear all fields.
